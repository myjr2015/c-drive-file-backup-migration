官方定位

窗口壳

多窗口、标题栏、拖拽区域、窗口尺寸、系统托盘。

命令桥

前端调用 Rust invoke，再执行本机文件操作。

权限模型

文件、Shell、Dialog、HTTP 等能力需显式授权。

字体
