Component Gallery

UI 组件图库

这个目录用于看每个开源 UI 项目“到底有多少组件、默认风格长什么样、主题和布局能怎么调”。它和业务界面演示分开，避免不同框架被同一套业务页面抹平差异。

打开方式

先看组件

从左侧选择一个框架，看按钮、输入、导航、数据展示、弹窗反馈和布局容器。

再看业务页

回到 docs/github_style_gallery.html 看它套进备份软件后的整体感觉。

最后定方案

选定框架和组件风格后，再迁移正式 app_fluent.py，不要边猜边改。
