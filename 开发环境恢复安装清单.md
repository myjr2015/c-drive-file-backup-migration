# 开发环境恢复安装清单

本文档用于 Windows 恢复系统后，重新安装常用编程语言、SDK、CLI 和本机开发依赖。目标是尽量把可移动/可备份的环境放在 D 盘，减少 C 盘恢复后重复配置。

## 总原则

- 优先安装到 D 盘：
  - `D:\code\YuYan\`
  - `D:\code\KuangJia\`
  - `D:\code\DaiMa\`
- C 盘只保留不可避免的用户配置，例如：
  - `C:\Users\<用户>\.codex`
  - `C:\Users\<用户>\.happy`
  - `C:\Users\<用户>\.ssh`
  - `C:\Users\<用户>\AppData\Roaming\npm`
- 恢复系统前后优先使用：
  - `D:\code\DaiMa\C盘备份软件`
  - 备份目录：`D:\code\DaiMa\#全局备份\AI会话备份`
- 安装完成后统一验证版本命令，并把可用结果记录到项目记忆。
